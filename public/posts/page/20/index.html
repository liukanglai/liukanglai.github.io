N个苹果放M个盘（N，M），允许有空。种类 思路： N<M，则一定有空盘，相当于（N，N），返回第二种情况即可。 N>=M，用递归的思想，分成放满M与不放满M 放满M等同于返回（N-M，M），即先保证放满 不放满就是返回（N，M-1），因为M-1的情况，又可分为放满与不放满，所以所有情况都讨论到了。 注意两种结束条件，返回N-M时，可能为0，return 1；当M=1时，return 1. #include<stdio.h> int way(int n,int m) { if(n==0||m==1) return 1; if(n<m) return way(n,n); return way(n,m-1)+way(n-m,m); }; int main(void) { int t,n,m; scanf("%d",&t); while(t) { scanf("%d %d",&n,&m); printf("%d\n",way(n,m)); t--; }; return 0; }
