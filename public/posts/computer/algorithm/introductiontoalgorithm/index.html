  1. master method
    • 特定的递归：T(n) = aT(n/b) + f(n), a >= 1, b > 1, n >= n0 with f(n) > 0;
    1. compare f(n) with n^logb(a)(树叶节点)
      • f(n) = O(n^(logb(a)-c)), c > 0; T(n) = Θ(n^logb(a))
      • f(n) = Θ(n^logb(a) _ log2n ^ k), k >= 0; T(n) = Θ(n^logb(a) _ log2n ^(k+1))
      • f(n) = Ω(n^logb(a) + c), c > 0, af(n/b) <= (1-c0)f(n), c0 > 0 上一层大于下一层; T(n) = Θ(f(n))

sorting#

  1. insertion sort algorithm

    for j <- [1-n];
    key = a[j];
    i = j - 1;
    while(i >= 0)
        a[i+1] = a[i];
        if(a[i--] < key)
            break;
    a[i+1] = key;

    while(i >= 0 && a[i] > key)
        a[i+1] = a[i];
        i--;
    a[i+1] = key;
  • θ(n^2)

  1. merge sort

    • n = 1, break; θ(1) just a constant

    • recursively sort a[0, n/2(向上取整)] and a[n/2 + 1(向下取整), n]; 2T(n/2)

    • merge subroutine. θ(n)

    • conquer: 比较两数组的第一项，谁小入谁，接着比.

  • T(n) = 2T(n/2) + cn = 2(2T(n/4) +cn/2) + cn = …
  • recursive tree, height is lg2n, leaves are n
  • T(n) = (cn)lg2n + θ(n) = θ(nlg2n)

Fibonacci numbers#

  • f(n) = 0, n = 0
  • f(n) = 1, n = 1

  1. native recursive

    • Ω(φ^n); φ = (1 + $\sqrt{2}$)/2

  2. Bottom-up
