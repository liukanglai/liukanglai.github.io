vimdiff#

  • vimdiff *.md *.md
    • ]c [c to move
  • do/:diffget get the different content
  • dp/:diffput put the different content
  • :%diffget/:%diffput all file to copy
  • 手动修改需更新高亮 :diffupdate/:diffu
  • 可多个文件, 但无 do/dp :diffget/:diffput 需指定参数:ls!或文件名
