## install fonts

- 将字体解压缩到 ~/.local/share/fonts（或 usr/share/fonts，以在系统范围内安装字体）
- 执行 fc-cache -f -v

## IDE

- 转到 Preferences/Settings → Editor → Font ，然后从字体下拉列表中选择 JetBrains Mono。
- `https://www.jetbrains.com/zh-cn/lp/mono/#font-family`
